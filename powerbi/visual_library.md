# RetailPulse Visual Library

## Core Visual Recommendations
| Visual | Best Use | Notes |
| --- | --- | --- |
| Card | Single headline KPI | Use for revenue, orders, customers, profit, churn rate, and service level |
| KPI Card | KPI with trend and status | Best on executive and cockpit pages |
| Matrix | Detailed operational review | Use for inventory actions, customer segments, and forecast comparisons |
| Table | Exception lists and drillthrough | Best for top customers, products, risks, and recommendations |
| Line Chart | Time-series trend | Use for revenue, orders, forecast, churn, and service levels |
| Area Chart | Cumulative momentum | Use for rolling revenue, portfolio value, and forecast accumulation |
| Bar Chart | Ranking and comparison | Best for top products, countries, categories, and risk buckets |
| Ribbon Chart | Rank movement over time | Good for product mix and customer segment shifts |
| Treemap | Share of revenue or risk | Use for product mix, category mix, and action mix |
| Heatmap | Dense categorical comparison | Best for segment x cluster, risk x health, and warehouse x horizon |
| Scatter Plot | Relationship analysis | Great for churn vs LTV, recency vs frequency, and stock vs risk |
| Bubble Chart | Relationship with magnitude | Use on churn, risk, and product value pages |
| Waterfall | Bridge and variance analysis | Use for profit bridge, cost bridge, and variance vs target |
| Decomposition Tree | Root cause analysis | Good for executive drill-down and what-drives-what analysis |
| Key Influencers | Driver analysis | Best on churn, retention, and health pages |
| Gauge | Target progress | Use sparingly for service level, fill rate, and target achievement |
| Map | Geographic distribution | Use for countries and regional clusters |
| Filled Map | Relative country intensity | Good for sales and churn heat by geography |
| Donut | Simple composition | Use only when categories are few and labels are clear |
| Funnel | Stage conversion or retention funnel | Use for retention, churn, and recommendation stages |
| Sankey (custom) | Flow across states | Use for customer movement or demand flow only when a custom visual is justified |

## Usage Rules
- Prefer built-in visuals first; introduce custom visuals only when the story needs flow or special decomposition.
- Avoid overcrowding a page with more than one dense visual type.
- Use tables and matrices for operational detail, but keep them behind drillthrough or exception pages.
- Keep custom visuals off the executive summary unless they materially improve decision-making.
